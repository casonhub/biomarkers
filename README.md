# Biomarkers-flask
Requirements

To run this application, you need:

    Python 3.x
    Requirment.txt

git clone https://github.com/casonhub/biomarkers.git
cd biomarkers

python -m venv venv
source venv/bin/activate   # On Windows, use venv\Scripts\activate
pip install -r requirements.txt

python app.py
